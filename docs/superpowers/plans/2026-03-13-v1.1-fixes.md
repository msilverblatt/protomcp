### Task 2: Add middleware registration to Go SDK runner

**Files:**
- Modify: `sdk/go/protomcp/runner.go`
- Modify: `sdk/go/protomcp/tool.go` (if middleware registry needed)

The Go SDK runner currently sends `ToolListResponse` and `ReloadResponse` (handshake-complete) but never sends `RegisterMiddlewareRequest`. Python and TypeScript SDKs do this between those two steps. The Go SDK needs a middleware registration API and the runner needs to send registrations during handshake.

- [ ] **Step 1: Add middleware registration API to Go SDK** (similar to Python's `@middleware` decorator / TS's `middleware()` function)
- [ ] **Step 2: Update runner.go to send RegisterMiddlewareRequest messages after ToolListResponse**
- [ ] **Step 3: Handle MiddlewareIntercept messages in the main loop (dispatch to registered handlers)**
- [ ] **Step 4: Run `go test ./...` in sdk/go**
- [ ] **Step 5: Commit**
